Proyectos

Task List: Angular

Desarrollado con Angular 14, este proyecto es una lista de tareas que permite agregar, eliminar y editar tareas. Indepentemente de la página en la que se encuentre, la información se guarda en IndexedDB del navegador.

BackEnd: JAVA y PHP

JAVA es la base para prácticamente todos los tipos de aplicaciones de red, además del estándar global para desarrollar y distribuir aplicaciones móviles y embebidas, juegos, contenido basado en web y software de empresa.

PHP (acrónimo recursivo de PHP: Hypertext Preprocessor ) es un lenguaje de código abierto muy popular especialmente adecuado para el desarrollo web y que puede ser incrustado en HTML.
